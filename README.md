# Astitwo Pandey — Portfolio

A modern, responsive portfolio website for Astitwo Pandey, a penetration testing and cybersecurity specialist. Built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Flowbite React

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
